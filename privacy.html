隱私權政策與免責聲明

最後更新日期：2025 年 12 月 20 日

1. 關於本服務

TW Academic Radar（以下簡稱「本站」）是一個非營利的開源專案，旨在協助學術工作者更有效率地搜尋台灣各大專院校與研究機構的職缺資訊。本站程式碼公開託管於 GitHub。

2. 資料來源與正確性

本站所有職缺資訊均透過自動化程式（爬蟲）彙整自公開網路資訊，主要來源包括但不限於：

  • 教育部大專教師人才網 (TJN)
  • 國家科學及技術委員會 (NSTC)

本站僅提供資訊索引與連結，**不保證**資料的即時性、正確性或完整性。所有職缺的詳細內容、應徵方式與截止日期，請以各徵才單位的官方公告為準。

3. 個資收集聲明

本站 **不收集、不儲存、不處理** 任何使用者的個人資料。

  • 本站為靜態網頁，無後端資料庫儲存您的瀏覽紀錄。
  • 本站沒有會員系統，不需要註冊或登入。
  • 本站不會要求您輸入姓名、電話、Email 等敏感個資。

4. Cookie 與追蹤

本站託管於 GitHub Pages，GitHub 可能會收集匿名的伺服器訪問日誌（Access Logs）以維護服務穩定性。此外，本站未主動安裝任何第三方廣告追蹤 Cookie。

5. 聯絡作者

若您對本站有任何建議，或發現資料有誤、希望撤下特定職缺資訊，歡迎透過 GitHub Repository 提交 Issue，或聯繫作者。
